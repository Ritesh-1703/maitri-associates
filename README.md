🏗 Maitri Associates – Engineering Services Website

This is the official corporate portfolio website for Maitri Associates Engineering Services – Civil Engineer & Contractor, built using modern web technologies.

The website showcases the company’s services, completed projects, vision, mission, and contact information in a professional and responsive design.

🚀 Tech Stack

Next.js (App Router)

TypeScript

Tailwind CSS

Framer Motion (Animations)

Lucide React Icons

Vercel (Deployment)

✨ Features

Modern corporate UI design

Fully responsive (Mobile, Tablet, Desktop)

Smooth animations using Framer Motion

Sticky animated navbar

Scroll progress indicator

WhatsApp floating contact button

Google Maps integration

SEO optimized (Metadata, Sitemap, Robots.txt)

Data-driven architecture (No hardcoding)

Optimized images with Next.js

🌐 Deployment

The project is deployed using Vercel for fast and reliable hosting.

📈 Purpose

This project was developed as a professional portfolio website for a construction and civil engineering company to showcase expertise, projects, and services publicly without requiring a backend database.

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
